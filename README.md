maven-repository
================

A personal maven repository for artifacts not tracked in a central repository.

Usage
-----
Add the following to your **pom.xml** file.
```xml
<repositories>
  <repository>
    <id>aparsons-github</id>
    <url>https://github.com/aparsons/maven-repository/raw/master</url>
  </repository>
</repositories>
```
